# 3D Chess

A fully playable 3D Chess game built in Java featuring AI opponents, multiplayer support, sound effects, and game-saving functionality.

## Features

- ♟️ Complete chess gameplay
- 🤖 Play against AI bots with selectable difficulty
- 🌐 Multiplayer support
- 💾 Save and load games
- 🔊 Sound effects for moves, captures, check, and checkmate
- 👑 Pawn promotion system
- 🏁 End-game overlay and victory detection
- 🎨 Custom chess piece graphics

## Technologies Used

- Java
- Java Swing / Graphics
- Object-Oriented Programming
- Networking (Client/Server)
- Audio Handling

## Main Components

### Bot System
- Chess AI implementation
- Multiple difficulty levels
- Move simulation

### Multiplayer
- Host and join games
- Client-server networking
- Move synchronization

### Game Management
- Turn handling
- Check and checkmate detection
- Save and load support

### User Interface
- Start menu
- Bot selection screen
- Promotion menu
- End-game overlay
- Sidebar display

## Controls

- Select a piece with the mouse.
- Click a valid square to move.
- Promote pawns when prompted.
- Use menu options to start local, bot, or multiplayer games.
